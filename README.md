# EELAR APP

Projeto EELAR com configurações importantes para começar novos aplicativos em React Native.

### Como instalar

A instalação apenas requere a instalação das dependências e a mudança no nome do projeto com o react-native-rename.

* Instalação das dependências
```
npm install
```

* Mudança do nome do projeto
```
npm install react-native-rename -g
react-native-rename <newName> -b <bundleIdentifier>
```
